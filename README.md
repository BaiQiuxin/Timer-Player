### Quick Start

#### Environment

1. Python 3.14
2. PyQt5

#### Install

1. Clone the repo
2. Run `.\run.bat`

```sh
git clone https://github.com/BaiQiuxin/Timer-Player.git
```

### File Tree

eg:

```sh

filetree 
├── LICENSE
├── README.md
├── player.py
├── initialize.py
├── /data/
│   └── /song/
│       ├── song1.mp3
│       └── song2.mp3
├── run.bat
├── timer_text.txt
├── player.ini
├── .gitignore
└── images
    └── logo.png


```
