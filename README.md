### Quick Start

#### Environment

1. Python 3.14
2. PyQt5 5.15.11

#### Install

1. Clone the repo
2. Put your audio files in `./data/song` and album cover images with the same names as your audio files in `./data/thumbnail`
3. Run `.\run.bat`

```sh
git clone https://github.com/BaiQiuxin/Timer-Player.git
```

### File Tree

eg:

```sh

MusicPlayer
├─ initialize.py
├─ LICENSE
├─ player.ini
├─ player.py
├─ README.md
├─ run.bat
├─ timer_text.txt
├─ images
│  └─ logo.png
└─ data
   ├─ thumbnail
   │  ├─ song1.png
   │  └─ song2.png
   └─ song
      ├─ song1.mp3
      └─ song2.mp3

```
